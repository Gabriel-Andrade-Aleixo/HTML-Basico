Guia de estilo HTML

Um código HTML consistente, limpo e organizado torna mais fácil para outras pessoas lerem e compreenderem seu código.

Aqui estão algumas diretrizes e dicas para criar um bom código HTML.



Use nomes de elementos em letras minúsculas

HTML permite misturar letras maiúsculas e minúsculas nos nomes dos elementos.

No entanto, recomendamos usar nomes de elementos em letras minúsculas porque:

  • Misturar nomes maiúsculos e minúsculos parece ruim
  • Os desenvolvedores normalmente usam nomes em minúsculas
  • Letras minúsculas parecem mais limpas
  • Letras minúsculas são mais fáceis de escrever

Exemplos:

Bom:

<body> <p>This is a paragraph.</p> </body>

Ruim:

<BODY> <P>This is a paragraph.</P> </BODY>

Fechar todos os elementos HTML

Em HTML, você não precisa fechar todos os elementos (por exemplo, o &lrp>elemento).

No entanto, recomendamos fortemente fechar todos os elementos HTML, assim:

Exemplos:

Bom:

<section>
<p>This is a paragraph.</p>
<p>This is a paragraph.</p>
</section>

Ruim:

<section> <p>This is a paragraph. <p>This is a paragraph. </section>
