Gráficos de tela HTML

O elemento HTML <canvas>é usado para desenhar gráficos em uma página da web.

O gráfico à esquerda é criado com <canvas>. Ele mostra quatro elementos: um retângulo vermelho, um retângulo gradiente, um retângulo multicolorido e um texto multicolorido.



Exemplos de tela

Uma tela é uma área retangular em uma página HTML. Por padrão, uma tela não tem borda nem conteúdo.

A marcação fica assim:

<canvas id="myCanvas" width="200" height="100"></canvas>

Adicione um JavaScript

Depois de criar a área retangular da tela, você deve adicionar um JavaScript para fazer o desenho.

Exemplo:

<script>
var c = document.getElementById("myCanvas");
var ctx = c.getContext("2d");
ctx.moveTo(0, 0);
ctx.lineTo(200, 100);
ctx.stroke();
</script>

Execução:

Your browser does not support the HTML canvas tag.

Desenhe um texto

Exemplo:

<script>
var c = document.getElementById("myCanvas");
var ctx = c.getContext("2d");
ctx.font = "30px Arial";
ctx.fillText("Hello World", 10, 50);
</script>

Execução:

Your browser does not support the HTML canvas tag.
