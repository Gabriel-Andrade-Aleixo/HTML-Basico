Elementos de formatação HTML

  • <b>- Texto em negrito
  • <strong>- Texto importante
  • <i>- Texto em itálico
  • <em>- Texto enfatizado
  • <mark>- Texto marcado
  • <small>- Texto menor
  • <del>- Texto excluído
  • <ins>- Texto inserido
  • <sub>- Texto subscrito
  • <sup>- Texto sobrescrito


Elemento HTML <strong>

O elemento HTML <strong>define texto com grande importância. O conteúdo interno normalmente é exibido em negrito.

Exemplo:

<strong>This text is important! </strong>

Execução

This text is important!

Elementos HTML <i> e <em>

O elemento HTML <i>define uma parte do texto com uma voz ou humor alternativo. O conteúdo interno normalmente é exibido em itálico.

Exemplo:

<i>This text is italic </i>

Execução:

This text is italic

O elemento HTML <em>define o texto enfatizado. O conteúdo interno normalmente é exibido em itálico.

Exemplo:

<em>This text is emphasized </em>

Execução:

This text is emphasized
