Estilos HTML

O atributo HTML "style" é usado para adicionar estilos a um elemento, como cor, fonte, tamanho e muito mais.



O atributo de estilo HTML

Definir o estilo de um elemento HTML pode ser feito com o styleatributo.

O atributo HTML stylepossui a seguinte sintaxe:

Exemplo:

<tagname style="property:value;">

A property é uma propriedade CSS. O value é um valor CSS.

Cor de fundo

A background-color propriedade CSS define a cor de fundo de um elemento HTML.

Exemplo:

Defina a cor de fundo de uma página como azul claro:

<body style="background-color:powderblue;">

<h1>This is a heading </h1>
<p>This is a paragraph. </p>

</body>

Execução:

This is a heading

This is a paragraph.

Exemplo:

Defina a cor de fundo para dois elementos diferentes:

<body>

<h1 style="background-color:powderblue;">This is a heading </h1>
<p style="background-color:tomato;">This is a paragraph. </p>

</body>

Execução:

This is a heading

This is a paragraph.

Cor do texto

A colorpropriedade CSS define a cor do texto para um elemento HTML:

Exemplo:

Defina a cor de fundo para dois elementos diferentes:

<h1 style="color:blue;">This is a heading </h1>
<p style="color:red;">This is a paragraph. </p>

Execução:

This is a heading

This is a paragraph.
